--- name: verifytxoutproof version: 0.22.0 group: blockchain permalink: doc/0.22.0/rpc/blockchain/verifytxoutproof/ --- verifytxoutproof "proof" Verifies that a proof points to a transaction in a block, returning the transaction it commits to and throwing an RPC error if the block is not in our best chain Arguments: 1. proof (string, required) The hex-encoded proof generated by gettxoutproof Result: ["txid"] (array, strings) The txid(s) which the proof commits to, or empty array if the proof can not be validated.
